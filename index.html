1. selectors

There are 3 selectors for css & html:

  • Element selector: element{} - change all element tags
    • For example:
      h1{} -- to change all header1 texts
  • Class selector: .classname{} - reusable
    • For example:
      .text{} -- to change all class named "text"
  • ID selector: #idname{} - unique
    • For example:
      #text{} -- to change all elements that has an ID of "text"

2. Basics in css

  • color: change text color
  • background-color: change background color
  • font-size: change size
  • font-family: the font, then the type of font
    • For example:

          color: red;
          background-color: yellow;
          font-size: 30px;
          font-family: serif, sans-serif;

3. Align

  • Align: where to put the text
    • For example:

      •     text-align: right;

4. spacing

  • margin: space outside element
    • For example:

      •     margin: 40px;

  • padding: space inside element
    • For example:

      •     padding: 40px;

5. border

  • border: to draw border (put size, type, and color)
    • For example:

      • border: 5px, dotted, blue

  • border-radius: round border (put size)
