1. Selectors

There are 3 selectors for css & html:

Element selector: element{} - change all element tags
For example:
h1{} -- to change all header1 texts

Class selector: .classname{} - reusable
For example:
.text{} -- to change all class named "text"

ID selector: #idname{} - unique
For example:
#text{} -- to change all elements that has an ID of "text"
